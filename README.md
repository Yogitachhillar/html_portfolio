# Yogita Chhillar — Data Analyst Portfolio

A fully static, single-file portfolio site built with vanilla HTML and CSS.

## Features
- Sticky nav with active section highlight
- Animated skill proficiency bars (scroll-triggered)
- Project showcase with tech tags and impact metrics
- Timeline-based experience and education section
- Contact section with direct links
- Responsive down to mobile
- Dark theme, no frameworks or build tools

## Stack
HTML · CSS · Vanilla JS · Tabler Icons · Google Fonts (Inter + DM Mono)

## Live site
[yogitachhillar.netlify.app](https://yogitachhillar.netlify.app) <!-- update this link -->

## Run locally
Just open `index.html` in any browser — no install needed.
